# Zepto SQL Data Analysis Project

## Project Overview
This project analyzes the Zepto Inventory Dataset using SQL to generate business insights.

## Objectives
- Analyze product categories
- Identify high-discount products
- Find out-of-stock items
- Calculate inventory value
- Analyze pricing and revenue opportunities

## Dataset
- Zepto Inventory Dataset

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE WHEN
- JOINS
- Subqueries
- CTEs
- Window Functions

## Business Questions Solved
- Products available in each category
- Highest discount products
- Out-of-stock products
- Average discount by category
- Most expensive products
- Revenue opportunity analysis
- Inventory analysis
- Ranking products using Window Functions

## Tools Used
- SQL
- MySQL
- GitHub
